# Закупки Максим — Оптимизация

Проект оптимизации управления закупочными ценами для ресторанной сети «Максим» (Тюмень).

## Что внутри

- `prototype/` — кликабельный HTML-прототип веб-системы (для демо заказчику)
- `legacy-apps-script/` — текущий код Apps Script из Google Sheets (исходники для миграции)

## Запуск прототипа

```bash
cd prototype
python3 -m http.server 8765
```

Открыть http://localhost:8765/

## Контур решения

Поставщики заполняют матрицы цен в Google Sheets → система автоматически собирает Топ-2 → формирует рекомендации шефам → сверяет с фактом из iiko/StoreHouse → считает переплаты и отклонения → AI-сводка для CEO.
