# Деплой на Timeweb Cloud Apps (Россия)

Российский DC (Москва/СПб), оплата в рублях, ~200–400₽/мес.
Деплой из GitHub-репо через Dockerfile (он уже в корне).

## Шаг 1. Регистрация (3 минуты)

1. https://timeweb.cloud → **Войти** → **Регистрация**
2. Email + пароль, подтвердить email
3. Привязать карту РФ (или пополнить баланс с любого кошелька)

## Шаг 2. Создать приложение (5 минут)

1. В кабинете → **Облачные приложения** (или Cloud Apps)
2. **Создать приложение** → **Из GitHub**
3. Подключить GitHub-аккаунт (auth-флоу один раз)
4. Выбрать репо `zamkofde-svg/maxim-zakup`, ветка `main`
5. Тип сборки: **Dockerfile** (Timeweb должен найти `Dockerfile` в корне сам)
6. Тариф: минимальный (обычно 1 CPU / 1 ГБ RAM, ~200–250₽/мес)
7. Регион: ближайший (Москва)

## Шаг 3. Переменные окружения (2 минуты)

В настройках приложения → **Переменные окружения** → добавить две:

### `OPENROUTER_API_KEY`
Значение скопировать командой:
```bash
cat ~/.config/maxim-zakup/openrouter.env
# взять то что после =
```

### `GOOGLE_SA_JSON_CONTENT`
Содержимое sa.json одной строкой:
```bash
cat ~/.config/maxim-zakup/sa.json
# скопировать ВЕСЬ JSON включая фигурные скобки
```

(После теста — оба ключа ротировать в их сервисах.)

## Шаг 4. Запуск

После сохранения переменных Timeweb автоматически билдит образ из
Dockerfile и стартует контейнер. **Билд ~5–7 минут.**

Когда статус **Запущено** → откроется ссылка вида
`https://maxim-zakup-XXX.twc1.net` (или твой кастомный домен).

При первом открытии БД пустая → авто-sync из Drive (заложено в коде).
Подожди 30–60 сек, обнови — данные появятся.

## Шаг 5. Кастомный домен (опционально)

Если хочется красивую ссылку (например `zakupki.maxim-rest.ru`):
- В настройках приложения → **Домены** → добавить → следовать инструкции
  (нужно прописать CNAME у регистратора).

## Шаг 6. Поделиться с заказчиком

Эту ссылку отдаёшь. Никаких VPN не нужно — DC в России.

## Автодеплой при git push

Включается одной галкой в настройках приложения **«Автоматический деплой»**.
После — любой `git push origin main` автоматически перебилдит и
обновит приложение через ~5 минут.

## Если упадёт

Timeweb → твоё приложение → **Логи** — там видно билд-логи и runtime.
Скинь скрин ошибки в чат — починим.

## Особенности

- **Не засыпает** (в отличие от Render free) — пользователь всегда
  получает ответ за <1 сек.
- **SQLite внутри контейнера** — при пересборке данные теряются и
  пересоздаются через Drive sync. Это нормально для MVP.
  Если потом понадобится сохранение причин отклонений между деплоями
  — подключим Timeweb PostgreSQL (~300₽/мес).
- **Лимит OpenRouter из РФ-DC** — если будет падать, переключу на
  YandexGPT/GigaChat (~30 минут работы).
